# Genetic Algorithms: Theory and Applications

Mathematical Notes, Implementations, and Experiments in Evolutionary Computation

## Overview

This repository contains my personal notes, theoretical developments, and practical implementations related to Genetic Algorithms (GAs) and Evolutionary Computation.

The project combines rigorous mathematical foundations with computational experiments, covering both the classical Genetic Algorithm framework introduced by Holland and modern extensions such as multimodal optimization, fitness sharing, and Markov chain analysis.

## Topics Covered

### Foundations of Genetic Algorithms

* Chromosomes and search spaces
* Genotype–phenotype mappings
* Fitness functions and fitness landscapes
* Population models
* Selection operators
* Crossover operators
* Mutation operators
* Canonical Genetic Algorithm (CGA)

### Theoretical Analysis

* Holland's Schema Theorem
* Building Block Hypothesis
* Implicit Parallelism
* Convergence Analysis
* No Free Lunch Theorem
* Population Dynamics

### Genetic Algorithms as Markov Chains

* Formal Markovian formulation
* Transition kernels
* Irreducibility
* Aperiodicity
* Ergodicity
* Stationary distributions
* Spectral gap analysis
* Mixing times
* First-hitting times
* Convergence to the global optimum

### Advanced Topics

* Evolution Strategies
* Multiobjective Optimization
* Pareto Fronts
* Estimation of Distribution Algorithms (EDA)
* Multimodal Genetic Algorithms
* Niching Methods
* Fitness Sharing
* Crowding Methods
* Gibbs Mixture Models
